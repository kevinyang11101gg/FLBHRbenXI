# Python英语四级单词在线学习系统

## 核心功能
- 单词发音：调用浏览器语音合成API即时朗读
- 释义显隐一键切换：先背再核对
- 标记功能：已掌握/困难/收藏三态标记，颜色+图标区分
- 本地JSON自动读写：含词条、学习位置、状态、统计，重启不丢
- 实时仪表盘：总词量、已学量、掌握率、困难率、收藏量、百分比进度条、分类详情可展开

## 适用场景
- 英语四级备考者
- 英语学习者
- 需要记忆单词的用户

## 技术特点
- 使用Flask框架进行开发
- 数据库使用MySQL
- 前端界面采用HTML、CSS和JavaScript构建
